# Vuh Stencil Components

## Install Package

```bash
npm install --save @the-ksquare-group/vuh-stencil-components
```

## Usage

1. Go to `/core` folder.

2. Install dependencies

   ```bash
   npm install
   ```

3. Run project

   ```bash
   npm start
   ```
